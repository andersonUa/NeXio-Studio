# NeXio-Studio

## Professional Interior Design & Architecture Platform

NeXio-Studio is a modern Flutter application created for interior designers, architects, and design studios.

## Main Features

- Project management
- Client database
- Interior planning
- Drawing management
- Measurements
- Material tracking
- Budget management
- PDF reports
- Calendar and reminders
- Photo gallery
- Offline functionality

## Professional Tools
